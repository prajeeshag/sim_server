Do not add a summary section, bullet list, or recap at the end of responses.

**Why:** User finds trailing summaries unnecessary — they can read the diff/output directly.

**How to apply:** End responses after the last meaningful action or inline explanation. No "Here's what was created/changed" wrap-up blocks.
